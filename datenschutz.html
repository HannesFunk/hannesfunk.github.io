Datenschutz bei DiRueLei

🔒 Vollständige Offline-Verarbeitung

DiRueLei ist eine Web-Anwendung, die vollständig im Browser ausgeführt wird. Alle Daten werden ausschließlich lokal auf Ihrem Gerät verarbeitet.

  • Keine Server-Uploads: Weder CSV-Dateien noch PDF-Dokumente werden auf einen Server hochgeladen.
  • Keine Cloud-Speicherung: Alle Verarbeitungsschritte finden lokal im Browser statt.
  • Keine externe Datenübertragung: Es werden keine personenbezogenen Daten an Dritte weitergegeben.
  • Keine Tracking-Cookies: DiRueLei verwendet keine Tracking- oder Analyse-Tools.

💼 Empfehlung: Nutzung auf Dienstgeräten

Wichtig: Lehrkräfte sollten DiRueLei ausschließlich auf schulischen Dienstgeräten verwenden.

  • Private Geräte sollten nicht für die Verarbeitung von Schülerdaten verwendet werden.
  • Stellen Sie sicher, dass das verwendete Dienstgerät den schulischen IT-Sicherheitsrichtlinien entspricht.
  • Nach der Nutzung können alle erzeugten Dateien lokal gelöscht werden.

📋 Welche Daten werden verarbeitet?

DiRueLei verarbeitet folgende Daten lokal in Ihrem Browser:

  • CSV-Dateien: Schüler-IDs und Namen aus Mebis-Exporten.
  • PDF-Dateien: Gescannte Dokumente mit QR-Codes.
  • QR-Codes: Automatisch generierte Identifikationscodes.

Alle diese Daten bleiben ausschließlich auf Ihrem Gerät und werden nicht übertragen.

🗑️ Datenlöschung

Da DiRueLei keine Daten speichert oder überträgt, können Sie jederzeit:

  • Den Browser-Tab schließen (alle Daten werden aus dem Arbeitsspeicher entfernt).
  • Die erzeugten PDF-Dateien lokal von Ihrem Gerät löschen.
  • Den Browser-Cache leeren, falls gewünscht.

🔐 Technische Sicherheit

  • Moderne Browser-Sicherheitsmechanismen (Cross-Origin Isolation).
  • Keine Persistierung von Daten über die Sitzung hinaus.
  • Verwendung von WebAssembly (Pyodide) für sichere Python-Ausführung im Browser.
  • Keine Netzwerkverbindungen während der Datenverarbeitung (außer initialem Laden der Anwendung).

⚖️ Rechtliche Grundlagen

Die Verarbeitung von Schülerdaten unterliegt den Bestimmungen der DSGVO und den jeweiligen Landesdatenschutzgesetzen. Die vollständige Offline-Verarbeitung von DiRueLei minimiert datenschutzrechtliche Risiken erheblich.

Hinweis: Diese Anwendung ist ein Werkzeug zur Unterstützung von Lehrkräften. Die Verantwortung für die datenschutzkonforme Verwendung liegt bei der nutzenden Lehrkraft bzw. der Schule.
